# 🧠 Interactive Quiz App

An interactive and responsive Quiz Application built using **HTML**, **CSS**, and **JavaScript**. The application presents multiple-choice questions, provides instant feedback, calculates the final score, and allows users to restart the quiz.

## 🚀 Features

- 📋 Multiple-choice quiz questions
- ✅ Instant correct/wrong answer feedback
- 📊 Live question progress
- 🏆 Final score display
- 🔄 Restart Quiz functionality
- 📱 Responsive design
- 🎨 Modern and clean user interface
- ⚡ Dynamic content updates using JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## ▶️ How to Run

1. Download or clone this repository.

```
git clone https://github.com/your-username/Interactive-Quiz-App.git
```

2. Open the project folder.

3. Double-click **index.html**
   or
   Open it with **Live Server** in Visual Studio Code.

4. Start answering the quiz questions.

## 📖 How It Works

- The quiz displays one question at a time.
- Select an answer.
- Correct answers are highlighted in green.
- Incorrect answers are highlighted in red.
- Click **Next** to move to the next question.
- After completing all questions, your final score is displayed.
- Click **Restart Quiz** to play again.
